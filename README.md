# Memory Match Game

A card matching memory game where players find pairs of cards.

## How to Play

1. Click on cards to flip them
2. Find two matching cards
3. Complete all pairs with minimum moves and time

## Project Structure

- `index.html` - Main game file
- `assets/css/style.css` - Styles
- `assets/js/main.js` - Game logic

## Development

To run locally, open `index.html` in a web browser.
